Секреты JavaScript ниндзя. Джон Резиг, Беэр Бибо, Иосип Марас Первое издание книги вышло в 2012 году, второе — в 2017. Я читала второе издание. В нём на подробных примерах авторы рассматривают методики и понятия языка JS. Обучение происходит от азов к мастерству. По задумке авторов читатели после прочтения должны стать хорошими специалистами.
